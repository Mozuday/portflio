## Features

* Responsive design
* Modern UI/UX
* Projects showcase
* Skills section
* About section
* Contact section
* Fast performance with Vite

* Vite
* Tailwind CSS

## Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate to the project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
